Example

This is a jekyll example page which shows all the variables you can use

Variables

  • * layout: default, blogpost, event, work
  • * title: The <title> of the page
  • * rooturl: Used to check current active page and set as body class
  • extra_css: Need extra styling on a page or post? Add 1 file here
  • extra_js: Need extra scripts on a page or post? Add 1 file here
  • extra_head_markup: Add markup right before the closing <head>
  • extra_footer_markup: Add markup right before the closing <body>

* mandatory
