Police Web Platform

An open government platform specifically built for the websites of the Belgian Local Police. It uses a component based architecture. Written in PHP, HTML, CSS and Javascript. Made by passionate open source technologists.

Why open matters?

We strongly believe that an open development model is the only way to foster badly-needed government standards, remove the fear of proprietary lock-in for government agencies, and create a large ecosystem that spans the government as a whole.

All of our code is available under an open source license.
